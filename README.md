# MBTI人格测试网站

一个完整的MBTI人格测试网站，包含60道专业测试题目，用户注册登录系统，以及详细的测试结果分析。

## ✨ 功能特点

- **60道标准测试题** - 基于经典MBTI理论体系，覆盖四个核心维度
- **用户认证系统** - 支持注册、登录，永久保存测试记录
- **实时进度显示** - 测试过程中实时显示答题进度
- **详细结果分析** - 四个维度可视化展示，16种人格类型详细解读
- **职业建议** - 针对每种人格类型提供适合的职业领域
- **响应式设计** - 完美适配桌面端和移动端

## 🧠 MBTI四个维度

- **E (外向) - I (内向)** - 精力来源和社交倾向
- **S (实感) - N (直觉)** - 信息获取和认知方式
- **T (思考) - F (情感)** - 决策方式和价值判断
- **J (判断) - P (感知)** - 生活态度和行事风格

## 🎮 使用方法

1. **访问网站** - 打开首页，点击"开始测试"
2. **注册/登录** - 创建账号保存测试记录
3. **完成测试** - 回答60道题目，选择最符合你的选项
4. **查看结果** - 获得详细的人格类型分析报告
5. **重新测试** - 可随时重新测试

## 🔒 数据隐私

- 所有用户数据存储在浏览器本地 (localStorage)
- 密码使用 Base64 编码存储
- 不会上传任何数据到服务器
- 可随时清除浏览器数据

## 📱 浏览器兼容性

- Chrome (推荐)
- Firefox
- Safari
- Edge
- 移动端浏览器

## 🚀 部署到 GitHub Pages

1. Fork 本仓库
2. 进入仓库 Settings
3. 找到 Pages 选项
4. Source 选择 main 分支
5. 点击 Save，等待部署完成
6. 访问 `https://your-username.github.io/mbti-test/`
